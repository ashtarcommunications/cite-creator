Cite Creator

Default shortcuts use Alt on PC, Option on Mac.

ctrl+alt+c / ctrl+option+c - Copy the current cite, optionally including any selected text.

The letter grade on each cite is a rough metric of how confident you should be in its accuracy.

To manually modify parts of the cite, select text from the page and press:

ctrl+alt+1 / ctrl+option+1 - Author name
ctrl+alt+2 / ctrl+option+2 - Qualifications
ctrl+alt+3 / ctrl+option+3 - Date
ctrl+alt+4 / ctrl+option+4 - Title
ctrl+alt+5 / ctrl+option+5 - Publication
